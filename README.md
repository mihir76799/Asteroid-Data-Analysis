# Asteroid-Data-Analysis

This project analyses the distance and relative velocity of asteroids drifiting through outer space with respect to earth by taking the csv data from NASA's records of asteroids from 1960 to 2060 (predicted). I have cleaned the data through Pandas and then used matplotlib and seaborn libraries in Python to plot the graphs given below.
